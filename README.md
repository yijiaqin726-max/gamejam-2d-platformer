# gamejar-2d-platformer

Unity 2D platformer gamejam project.

## Collaboration Rules

- `main` must stay runnable.
- Work on personal development branches such as `dev/yijia` and `dev/programmer2`.
- Do not edit the same Scene or Prefab at the same time.
- Do not delete `.meta` files.
- Do not commit `Library/`, `Temp/`, `Obj/`, `Logs/`, or build output.
- Use Git LFS for large art and audio assets.

## Unity Required Settings

After creating or opening the Unity project, set:

- `Edit > Project Settings > Editor > Version Control Mode`: `Visible Meta Files`
- `Edit > Project Settings > Editor > Asset Serialization Mode`: `Force Text`

Before merging into `main`, confirm Unity opens without red console errors, the player can move, levels can load, and a build can be made.
